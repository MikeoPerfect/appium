## HTTP API 规范


### 路径

`POST /session/:session_id/appium/device/toggle_location_services`


### URL 参数

|名称|描述|
|----|-----------|
|session_id|新建会话(session)后用来标识当前会话(session)的ID|


### JSON 参数

无


### 响应

null
